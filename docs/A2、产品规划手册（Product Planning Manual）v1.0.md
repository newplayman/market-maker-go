# **1. 产品愿景（Vision）**

构建一个 **可长期稳定运行、具备专业风控、自动化执行、实时可监控、可扩展、可进化** 的加密货币做市商系统，用以：

- 稳定赚取交易手续费与 spread 收益
    
- 通过低延迟优势增加策略成交率
    
- 在无方向性暴露的前提下稳定提升资金利用效率
    
- 最终通过大规模自动化实现“睡后收入”
    

一句话：

> **打造一个稳定、安全、自动化、可扩展的做市商交易大脑。**

---

# **2. 产品定位（Positioning）**

本产品定位为：

### **轻量级高频做市系统（Lightweight HFT Market Maker System）**

特点：

- 持仓风险低
    
- 操作频率适中（几十~几百次/分钟）
    
- 利用免费 Maker费率、低延迟、智能挂单来赚钱
    
- 专注 Binance USDC-M 市场起步
    
- 数据驱动，可扩展至多交易所
    

该系统是：

- 不是趋势跟随
    
- 不是网格机器人
    
- 不是孤立 API bot  
    而是：
    

> **一个专业做市商的轻量版本（可升级到专业级）。**

---

# **3. 用户定位**

本产品主要面向 **量化交易开发者 / 自用交易团队**：

### ① 个人量化交易者（专业级）

具备编码能力，希望用自己的策略稳定增长资产。

### ② 小型量化团队

2~5 人团队，需要标准化系统架构、风控、OMS。

### ③ 自动交易平台进阶用户

未来可模块化开放策略框架，提供插件。

# **5. 产品核心价值**

系统的 5 大核心价值：

---

### **① 稳——风险优先体系**

核心不是收益，而是：

- 底层保证不爆仓
    
- 风控严格不越界
    
- 异常自保（panicstop）
    
- 自动对冲
    
- 环境自恢复
    

---

### **② 快——低延迟执行能力**

通过：

- 新加坡 VPS
    
- 0.2ms WS 延迟
    
- Go 语言执行
    
- 最小化策略计算逻辑
    

达到 **轻量 HFT 级执行速度（5–15ms）**。

---

### **③ 智——策略动态决策**

从固定挂单 → 动态做市商：

- 动态 spread
    
- 动态 inventory 风险控制
    
- 动态网格
    
- 订单簿结构驱动报价
    
- 微观结构偏移判断
    

提高收益与安全性。

---

### **④ 稳定运维（Ops）**

系统必须：

- 能跑一年不重启
    
- 有监控
    
- 有告警
    
- 有审计
    
- 有巡检
    
- 有自动恢复
    

这是专业系统的基础。

---

### **⑤ 可扩展（为未来 AI 做准备）**

架构可以容纳：

- 强化学习做市模型
    
- LLM 解释器
    
- 多模型协商
    
- 最优参数搜索
    

你未来的“多模型大脑”可直接接入。

# **8. MVP 范围（Minimal Viable Product）**

MVP 必须能：

- 24/7 自动运行
    
- 每天自动产生稳定 PnL
    
- 自恢复
    
- 自动撤单/平仓
    
- 自带可视化监控
    
- 可人工调整参数
    

MVP 不需要：

- AI
    
- 多交易所
    
- 高级策略
    
- 大数据系统
    

MVP 范围包含：

- Phase1/Phase2 策略
    
- OMS
    
- Gateway
    
- Windshield（WebUI 监控）
    
- 风控模块
    
- 日志/告警
    
- 单节点部署

# **10. 风险分析**

主要风险：

### ① 市场极端波动

- 使用 panicstop
    
- 使用快速对冲
    
- 自动撤单
    
- 限制 inventory
    

### ② 交易所 API 不稳定

- 高质量 Gateway
    
- 断线重连机制
    
- snapshot 恢复机制
    

### ③ 策略 bug

- 单测 + 仿真 + 灰度上线流程
    

### ④ 黑客攻击

- WebUI 权限体系
    
- API key 加密
    
- 2FA 登录
    

### ⑤ 技术债累积

- 完整文档
    
- 规范开发流程
    
- 代码审查
